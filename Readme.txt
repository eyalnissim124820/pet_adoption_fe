My Full-Stack pet adoption dashboard system (**desktop version only).

Deployment link: https://pet-adoption-fe-theta.vercel.app/

in order to run it on your computer:
1.Install Node.js.
2.For backend folder and frontend folder separately:
  2.1. Open terminal and cd in to the folder
  2.2. Run the command `npm install`
3.Open VS Code for backend folder
  3.1. open terminal inside the folder.
  3.2. run `nodemon server.js`
4.Open VS Code for backend folder
  4.1. open terminal inside the folder.
  4.2. run `npm start`
